a++ 与 ++a 的区别

HAO  收录于 C语言程序设计
2018-10-14  约 240 字  预计阅读 1 分钟   次阅读

  • a++ 是先返回百a的值，再执行++运算。
  • ++a 是先执行++运算，在返回a的值。

#include <stdio.h>

int main(int argc, char **argv) {
    int a=100;
    printf("a = %d\n",a);
    printf("a++ = %d\n",a++);
    printf("a = %d\n",a);
    printf("++a = %d\n",++a);
    printf("a = %d\n",a++);
    return 0;
}

a = 100
a++ = 100
a = 101
++a = 102
a = 102

++a和a++等价的结果一样，但是运算过程不同，a++ 是先使用a的值，然后再对a做加1处理，++a是先对a作加1处理，然后再使用a的值。
